# 移动端适配说明文档

## 概述
为表格表单中的批量导入弹框组件添加了完整的移动端适配，确保在移动设备上提供良好的用户体验。

## 适配的组件

### 1. ImportSteps.vue - 主导入步骤组件
**主要功能：**
- 自动检测移动端设备（屏幕宽度 ≤ 768px）
- 响应式对话框宽度（移动端：95%，桌面端：800px）
- 移动端优化的步骤指示器布局
- 触摸友好的按钮尺寸和布局

**移动端特性：**
- 垂直布局的步骤指示器
- 全宽按钮，高度44px，适合触摸操作
- 响应式内容区域padding
- 移动端优化的对话框边距和高度

### 2. StepSelectFile.vue - 文件选择步骤
**移动端优化：**
- 触摸友好的上传区域（高度150px，移动端）
- 移动端优化的提示文字（"点击选择文件"）
- 全宽下载按钮，44px高度
- 响应式提示信息布局
- 触摸设备悬停效果优化

**响应式断点：**
- 768px以下：中等移动端优化
- 480px以下：小屏移动端优化

### 3. StepPreviewData.vue - 数据预览步骤
**移动端特性：**
- 卡片式数据展示（替代传统表格）
- 移动端最多显示50条数据（桌面端100条）
- 触摸友好的工作表选择器
- 响应式布局和字体大小

**移动端布局：**
- 每行数据独立卡片
- 清晰的字段标签和值显示
- 滚动优化的内容区域
- 触摸友好的交互元素

### 4. StepImportSettings.vue - 导入设置步骤
**移动端优化：**
- 卡片式字段映射界面
- 触摸友好的下拉选择器
- 响应式验证信息显示
- 移动端优化的间距和字体

**移动端特性：**
- 每个Excel列独立卡片
- 清晰的标签和选择器布局
- 触摸友好的交互元素
- 响应式警告信息显示

### 5. StepImportData.vue - 导入结果步骤
**移动端适配：**
- 响应式图标尺寸（移动端80px，桌面端60px）
- 移动端优化的文字大小和间距
- 触摸友好的警告信息滚动区域
- 响应式内容布局

## 技术实现

### 移动端检测
```javascript
const isMobile = ref(false)

const checkMobile = () => {
  isMobile.value = window.innerWidth <= 768
}

onMounted(() => {
  checkMobile()
  window.addEventListener('resize', checkMobile)
})
```

### 响应式设计
- 使用CSS媒体查询适配不同屏幕尺寸
- 768px以下：平板和手机横屏
- 480px以下：手机竖屏
- 触摸设备优化（hover: none, pointer: coarse）

### 组件通信
- 通过props传递`isMobile`状态
- 子组件根据移动端状态调整UI
- 保持组件间的数据流一致性

## 使用说明

### 基本用法
```vue
<ImportSteps
  v-model="showImport"
  :columns="tableColumns"
  :table-title="tableName"
  :on-import="handleImport"
/>
```

### 移动端自动适配
组件会自动检测设备类型并应用相应的样式，无需额外配置。

### 自定义移动端样式
可以通过CSS变量或深度选择器自定义移动端样式：
```css
.mobile-dialog :deep(.el-dialog) {
  /* 自定义移动端对话框样式 */
}
```
